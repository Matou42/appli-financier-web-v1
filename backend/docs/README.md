# Finance Manager App

A web application to manage your finances, track transactions, and more.

## Features

- User authentication (signup, login, logout)
- Password reset functionality
- Dashboard to view financial summary
- Profile management
- Transaction management

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Node.js, Express
- **Database:** MySQL

## Getting Started

### Prerequisites

- Node.js (>=14.x)
- npm (>=6.x)
- MySQL Database

### Installation

Follow the steps in the [Installation Guide](INSTALL.md) to set up and run the application locally.
