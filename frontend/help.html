Aide - Gestion Financière

Démarrage rapide

Bienvenue dans l'aide de l'application Gestion Financière. Cette section vous guide à travers les étapes de base pour commencer à utiliser l'application.

  • Créer un compte : Pour utiliser toutes les fonctionnalités, vous devez d'abord créer un compte utilisateur.
  • Se connecter : Utilisez vos identifiants pour vous connecter à l'application.
  • Explorer le tableau de bord : Découvrez les informations principales affichées sur le tableau de bord.

Fonctionnalités principales

Découvrez les principales fonctionnalités offertes par l'application Gestion Financière :

  • Gestion des transactions : Ajouter, modifier ou supprimer des transactions financières.
  • Profil utilisateur : Modifier les informations de votre profil et vos préférences.
  • Paramètres : Personnaliser les paramètres de l'application selon vos besoins.

FAQ

Questions fréquemment posées sur l'utilisation de l'application :

Comment réinitialiser mon mot de passe ?
Pour réinitialiser votre mot de passe, cliquez sur "Mot de passe oublié" sur l'écran de connexion et suivez les instructions.
Comment exporter mes données ?
Allez dans les paramètres, puis sélectionnez l'option "Exporter mes données".
